# FinSight – Explainable Fintech Portfolio Intelligence

Live Demo  
https://finsight-lab.streamlit.app/

FinSight is an explainable fintech analytics platform that enables users to:

• Analyze stock performance with interactive visualizations  
• Backtest technical trading strategies  
• Optimize portfolios using Modern Portfolio Theory  
• Generate plain-English explanations for financial insights  

## Tech Stack
Python  
Streamlit  
Pandas  
NumPy  
yFinance  
Plotly  

## Features

### Stock Analytics
Visualize historical returns, moving averages, and performance vs benchmark.

### Strategy Backtesting
Compare simple trading strategies with Buy-and-Hold performance.

### Portfolio Optimization
Compute efficient portfolios using risk-return optimization.

### Explainable Insights
Translate complex quantitative results into human-readable insights.
